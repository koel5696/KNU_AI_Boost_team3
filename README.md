# KNU_AI_Boost_team3

공유 메일 내용을 정리하여 대학 부서 담당자의 홈페이지 공지 작성을 간편하게 돕는 1-Day PRD 기반 프로토타입입니다.

외부 공고 메일을 입력하면 유형, 대상, 기간, 주요 혜택, 신청 방법, 문의처를 규칙 기반으로 정리하고, 확인이 필요한 항목은 `담당자 확인 필요`로 표시합니다. 결과는 학교 홈페이지 게시글 형태로 복사할 수 있으며, 실제 AI 연동이 아닌 예시 결과임을 화면에 안내합니다.

## 실행 방법

```bash
npm install
npm run dev
```

빌드와 점검:

```bash
npm run build
npm run lint
npm run check:prototype
```

## 환경 변수

`.env.example`을 참고해 필요한 경우 `.env`를 만듭니다.

```env
VITE_APP_ENV=development
```

현재 프로토타입은 별도 API 키나 비밀 값을 필요로 하지 않습니다.

## 지원하는 입력

- 이메일 본문 직접 입력 및 `.eml` 파일
- 이미지: PNG, JPG, WEBP, BMP, HEIC/HEIF
- 문서: PDF, DOCX, XLSX, TXT, HTML
- 이메일에 포함된 지원 형식의 첨부파일
- 이미지와 스캔 PDF의 한글·영문 OCR
- 이미지와 PDF의 QR 코드 및 URL

파일은 서버에 업로드하거나 저장하지 않고 브라우저에서 처리합니다. 스캔 문서 OCR은 첫 실행 시 한글·영문 언어 데이터를 내려받으므로 인터넷 연결이 필요하고 시간이 걸릴 수 있습니다.

## 검토 기능

- 추출한 값의 파일명·PDF 페이지·원문 근거 표시
- OCR 여부를 반영한 필드별 신뢰도 표시
- 본문과 첨부파일 사이의 서로 다른 값 감지
- 추출 결과 직접 수정
- 필수 항목 검토를 확인한 뒤 홈페이지·SNS·메시지 초안 복사

암호화 문서, `.msg`, 구형 `.xls` 파일은 현재 지원하지 않으며 파일 목록에 오류가 표시됩니다. 파일은 최대 8개, 각 20MB까지 처리합니다.
